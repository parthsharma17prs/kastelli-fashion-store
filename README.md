# Kastelli Premium Fashion Store App

> Premium, high-fidelity mobile experience developed with modern React Native best practices.

## 🌟 Overview
A luxury e-commerce mobile application showcasing high-fidelity fashion retail experiences, smooth micro-interactions, dynamic cart synchronization, and premium visual layout standards.

## 🛠️ Technology Stack
- **Core Framework**: React Native, Expo SDK 49, TypeScript, TailwindCSS (NativeWind), Framer Motion / Reanimated, HSL Custom Palettes, outfit typography
- **Aesthetic Direction**: Sleek glassmorphism, HSL curated colors, fluid layout engine, custom typography.
- **Performance Focus**: Optimized render loops, pre-cached assets, strict memory management.

## 🚀 Key Features
- **Visual Excellence**: Smooth micro-animations on all interactive elements.
- **Dynamic Interaction**: Fluid list transitions and custom spring physics.
- **Premium Design System**: Complete adherence to modern high-end mobile layout aesthetics.

## 💎 Advanced Visual Aesthetics
- **Tactile Feedback**: Haptic/vibration patterns configured on core checkout and CTA triggers.
- ** Curated Colors**: Tailored HSL scheme designed for elegant contrasts under both light and dark mode ambient settings.
- **Micro-interactions**: Scale gestures (e.g. `0.98` scale down on press) for high-tactility, premium response.
- **Details**: Utilizes optimized FlashList components for endless product grids and pre-fetched image caching to maintain a fluid 120Hz frame rate.

## 🏁 Run & Installation Instructions
1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
